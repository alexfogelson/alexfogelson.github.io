A Quirkey Python Script: Leveraging the iTunes Database

Every so often, while perusing the internet for content offering me modicum amount of dopamine, I come across someone who has made a custom playlist, which when read as an acrostic (using the first word, rather than the first letter), will yield a particular message of sentence. On a whim, I decided to see if I could automated this process, solely because it would be entertaining and acknowledging the near certain lack of utility.

Now unfortunately, in order to actually create the playlist, one needs a developer account to utilize the API from music.apple.com But luckily, a free version is available, which allows one to search through all iTunes content, including movies, podcasts, apps, and books. About two hours later, the tool was complete!

Click here to download. Be sure to be running an updated version of python. To run, use the command python3 playlist_maker.py "Sentence here".

Dimensionality and Gravity: Polishing an Endevour from 9th grade

Years and years ago, as a freshman in highschool, I came accross a wonderful little bit of physics (admittedly a bit over my head at the time), about the inverse square law and it's relation to our 3-dimensional world. The article described the relation between gravitational flux and the inverse square, which results as a consequence of surface area relating to the square of lengths.

More specifically, it asserted that in a 2-dimensional world, the force of gravity would be simply proportional to the inverse of the distance, and similarly in a 4-dimensional world, it would be proportional to the inverse cube of the distance (four dimensional spheres have cubic flux analogs).

As an amateur Javascript learner, I decided my science fair project for the year would vary the number of dimensions, and note how Earth's orbit changed accordingly. My finished product used an online Javascript IDE, had no visual components, nor easily adjustable settings.

So recently, I decided to update the model from years ago, and include the components that are missing. To run the project below, you first need to install PyGames using pip3 install pygames. Then run the program with python3 gravitySimulator.py. The code can be downloaded here.
